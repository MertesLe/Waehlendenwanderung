# Waehlendenwanderung

Ziel dieses Projekts ist die Untersuchung von Wählerwanderungen zwischen den Bundestagswahlen 
2021 und 2025 auf möglichst kleinräumiger Ebene. Hierfür werden die amtlichen 
Wahlbezirksergebnisse beider Wahlen unter Berücksichtigung gemeinsamer Briefwahlauszählungen, 
gemeinsamer Auszählungen nach § 68 BWO sowie kommunaler Gebietsänderungen zu harmonisierten 
Gemeindeaggregationen zusammengeführt. Auf diesen Analyseeinheiten werden mittels nslphom 
lokale Wählerübergangsmatrizen geschätzt und anschließend Zusammenhänge zwischen 
Wählerwanderungen und regionalen sozioökonomischen bzw. demografischen Strukturmerkmalen 
untersucht.
